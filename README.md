# Medical App

## Overview
This is a web-based medical application developed to streamline healthcare management. The app allows users to manage patient information, appointments, prescriptions, and more. Additionally, it features a speech-to-text conversion tool, built using Python, that enables voice-based data input.

## Features
- **Patient Dashboard**: View and manage patient profiles and their associated medical data.
- **Appointment Management**: Schedule, update, or cancel patient appointments.
- **Prescription Management**: Easily create, view, and manage prescriptions.
- **Speech-to-Text Integration**: A Python-powered feature to convert spoken words into text for easier data entry.
- **Backend Powered by Go**: The backend is built using Golang for fast and efficient processing.

## Tech Stack

### Frontend
- **React**: User interface development for building dynamic web components.
- **CSS/SCSS**: For custom application styling.

### Backend
- **Go (Golang)**: Server-side logic and API management.

### Speech-to-Text Conversion
- **Python**: Used for the speech-to-text functionality, leveraging the `speech_recognition` library.
